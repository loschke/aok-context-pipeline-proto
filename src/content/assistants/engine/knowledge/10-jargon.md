# Jargon-Uebersetzung

## Fuer nicht-technisches Publikum

Bei Stakeholdern, AOK-Team und nicht-technischen Gespraechspartnern diese Begriffe uebersetzen:

| Interner Begriff | Stattdessen sagen | Erklaerung |
|---|---|---|
| Context Engineering | Wissensaufbereitung | Wie wir AOK-Inhalte fuer den Assistenten aufbereiten |
| Chunks / Contexts | Wissenseinheiten / Wissensbausteine | Die einzelnen Informationspakete |
| Taxonomie | Themenstruktur | Wie die Wissensbausteine sortiert sind |
| Retrieval | Wissensabruf | Wie der Assistent die richtige Information findet |
| Embedding | (nicht erwaehnen) | Technisches Detail, irrelevant fuer Verstaendnis |
| Vector Database | (nicht erwaehnen) | Technisches Detail |
| System-Prompt | Anweisungen / Regelwerk | Die Regeln, nach denen der Assistent arbeitet |
| Frontmatter | Metadaten / Beschreibungsfelder | Die Zusatzinformationen zu jedem Baustein |
| Pipeline | Aufbereitungsprozess | Die 8 Schritte der Transformation |
| Full Context Loading | Alles-auf-einmal-Laden | Alle Bausteine werden gleichzeitig geladen |
| MCP | Schnittstelle / Anbindung | Technisches Protokoll fuer Tool-Nutzung |
| Token | Texteinheit | Masseinheit fuer Textlaenge (ca. 0.75 Woerter) |
| Context Window | Arbeitsgedaechtnis | Wie viel Text der Assistent gleichzeitig "sehen" kann |
| Halluzination | Erfundene Antwort | Wenn der Assistent etwas plausibel klingendes erfindet |
| RAG | Wissensgestuetztes Antworten | System sucht erst Fakten, dann antwortet es |
| DGW | Allgemeine Gesundheitswelt (ohne Login) | Oeffentlich zugaenglicher Bereich |
| MGW | Meine Gesundheitswelt (mit Login) | Persoenlicher Bereich mit Versichertendaten |

## Wann Fachbegriffe erlaubt sind

Bei technischem Publikum (Entwickler, AI-Team) koennen Fachbegriffe frei verwendet werden. Der Gespraechspartner signalisiert das durch eigene Verwendung von Fachbegriffen.

Faustregel: Wenn der Nutzer "Embedding" sagt, darfst du "Embedding" sagen. Wenn der Nutzer "Wie findet der Assistent die richtige Antwort?" sagt, erklaere es ohne Fachbegriffe.
